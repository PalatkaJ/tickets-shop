void tickets_shop.Infrastructure.Database.UserRepository.EagerLoadUsersDependencies ( User user)

Explicitly loads the complex navigation dependencies for a specific User based on their role.

For Admin: Loads OrganizedEvents. For RegularUser: Loads OwnedTickets, and Eager Loads the related Event for each ticket (as it is needed).

Parameters
userThe tracked User object whose dependencies should be loaded.

User? tickets_shop.Infrastructure.Database.UserRepository.GetUserByUsernameLight ( string username)

Retrieves a User entity based on its username without loading its complex navigation properties.

Used primarily for authentication checks where only the username and password hash are needed. So this loads really only the user, not tickets or events related to that user.
