Detailed Description

A static service class that acts as a central repository for all menu builder instances.

It uses the Lazy pattern to ensure that menu objects are only initialized and constructed the very first time they are accessed, optimizing application startup.

Initializes all Lazy builders, setting up the necessary dependencies but deferring the actual object construction until the .Value property is first accessed.

This method runs only once during initialization.
